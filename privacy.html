Политика конфиденциальности

Мы уважаем вашу конфиденциальность и стремимся защитить ваши персональные данные. Данная политика объясняет, как AUM обрабатывает информацию.

1. Собираемые данные

Приложение AUM может собирать следующие данные:

  • Уникальный идентификатор оборудования (HWID) для привязки лицензии PRO.
  • IP-адрес при обращении к серверу обновлений.
  • Версия установленной игры и пути к игровым файлам (локально, не передается на сервер).

2. Использование данных

Ваш HWID используется исключительно для обеспечения работы платных функций и предотвращения пиратства. Мы не передаем ваши данные третьим лицам.

3. Хранение данных

Данные о лицензиях хранятся в защищенной базе данных на нашем сервере и удаляются по истечении срока действия ключа или по запросу пользователя.

4. Cookies и сайт

Наш сайт не использует куки для отслеживания рекламы. Мы можем использовать аналитику GitHub Pages для просмотра общего количества посещений.

5. Контакты

Если у вас есть вопросы по поводу ваших данных, свяжитесь с нами через официальный Telegram канал.
